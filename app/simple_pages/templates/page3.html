Testing with PyTest and Simple Pages

March 7, 2022 by Collin Fogerty

Testing with PyTest is a utility provided by Flask which allows for a user to confirm that certain words or phrases are in and working with their source code. This is valuable when a project has a lot of folders and code that could get hard to confirm everything is there. A line like this "content = bytes(test_string, 'utf-8')" will test that the environment is utf-8 and it checked that by making sure utf-8 is printed in the code.

Simple Pages were created using parts of templates from the BootStrap v.5. These are the pages that model the actual user interface of the website. These are tested in two ways. First there is a test that makes sure the page itself is named correctly. Secondly there is a test that checks that the content in the webpage matches the expected content based off the web page name.
